## Prerequisites

The following technologies are required for the project:

**Frontend:**
  - HTML5 / Bootstrap 5 (css)
  - JavaScript
  - React (v19+)
  - Next.js (v16+)
  - qrcode.react (v4.2.0)

**Backend:**
  - Spring Boot (v3.5.1) + JPA
  - Java JDK 21
    
**Database:**
  - H2 (development)
  - MySQL (production)
    
**IDE:**
  - Frontend: Visual Studio Code
  - Backend: Eclipse IDE for Java EE Developers

**Tools:**
  - Docker Desktop
